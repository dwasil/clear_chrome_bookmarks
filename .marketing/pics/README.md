# Marketing Image Concepts

5 promotional image concepts for the Dead Bookmark Finder Chrome extension.

## Quick Overview

| # | Concept | Theme | Mood |
|---|---------|-------|------|
| 1 | **Graveyard** | Halloween-fun, RIP dead links | Humorous, playful |
| 2 | **Spring Cleaning** | Sweeping away broken bookmarks | Fresh, satisfying |
| 3 | **Doctor** | Medical checkup for sick bookmarks | Caring, fun |
| 4 | **Superhero** | Fighting evil 404 monsters | Action, empowering |
| 5 | **Before/After** | Split-screen transformation | Satisfying, wow-effect |

## Brand Elements to Include

- **Icon:** Blue bookmark with magnifying glass & red X
- **Primary color:** #4A90A4 (blue)
- **Accent:** #E74C3C (red for errors/dead)
- **Success:** #2ECC71 (green for clean)

## Chrome Web Store Image Specs

- **Small promo tile:** 440 x 280 px
- **Large promo tile:** 920 x 680 px
- **Marquee:** 1400 x 560 px

## Recommended Tools for Image Generation

1. **Midjourney** - Best for stylized illustrations
2. **DALL-E 3** - Good for clean vector-style graphics
3. **Canva** - For adding text overlays and final touches
4. **Figma** - For compositing icon with generated backgrounds

## Usage

Each concept file contains:
- Detailed composition description
- Color palette with hex codes
- Ready-to-use AI image generation prompt
- Text overlay suggestions
